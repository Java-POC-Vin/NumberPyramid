## Number Pyramid

This project prints out in prompt a triangle built of a sequence from 1 to the desired number (n), repeating the number by itself!
this is a project to exercise multiple loops logic in Java.

## How to use

1. Go to ./src/App.java

2. Change the following variable value to a desired number of sequences:
```java
int num = 5;
```

3. Run the script.

## Output example

```java
// sequence example for num = 5;
1
22
333
4444
55555
```
